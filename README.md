# Rayan SPA — Sito vitrine

Sito web vetrina per **Rayan SPA**, una spa con hammam e massaggi nel quartiere
di **Dar El Bacha**, a Marrakech.

Il sito presenta l'atmosfera del luogo e i trattamenti offerti, con un'estetica
chiara e calda ispirata a Marrakech (terracotta e accenti rossi). I contenuti
sono in **francese**, pensati per una clientela internazionale e turistica.

Le richieste di prenotazione non passano da un server: il modulo compila un
messaggio già pronto e apre **WhatsApp** per inviarlo alla spa.

## Sezioni principali

- **Accueil** — presentazione e invito alla prenotazione
- **À propos** — la spa e la sua filosofia del benessere
- **Soins & Massages** — i trattamenti con durata e prezzo
- **Galerie** — immagini e atmosfera del luogo
- **Réservation** — modulo che apre WhatsApp
- **Témoignages** — recensioni dei clienti
- **Contact** — indirizzo, orari e posizione su mappa

## Avvio in locale

```bash
npm install
npm run dev
```

Il sito sarà disponibile su [http://localhost:3000](http://localhost:3000).
